# Library Book Tracking System

A console-based Java application for managing library books using JDBC and MySQL.

## Features
- Add and view books
- Issue and return books
- User-based book issuing
- Fine calculation using date difference
- CSV export of issue history
- Proper database transactions and validation

## Tech Stack
- Java
- JDBC
- MySQL
- Git & GitHub

## How to Run
1. Configure MySQL database
2. Update DB credentials in DBConnection.java
3. Run Main.java
